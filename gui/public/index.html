Architecture Stack

            

              
embedMiniLM-L6-v2 (384-d) → Random Proj → 64-d

              
+ 8 structural features → 72-d node input

              
sageSAGEConv(72 → 128) → ReLU → Dropout(0.25)

              
gatGATConv(128 → 32, heads=4, concat=False)

              
→ attention weights saved per edge (explainability)

              
headLinear(32 → 1) → Sigmoid → confidence [0,1]

Safety Override

            

              Even if the GNN confidence ≥ threshold, the hasSideEffect() check
              in analysis.cpp acts as a hard veto.
              Nodes with side_effect=1 are never removed by the ML pass.
              Nodes with confidence 0.50–0.60 are deferred (uncertainty zone).
